# Hasan Ibnu Ali Professional Portfolio

Professional portfolio website for Hasan Ibnu Ali, built as a dark premium editorial portfolio for recruiters, scholarship reviewers, and institutional reviewers.

## Stack

- React
- TypeScript
- Vite
- CSS

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Vercel Deployment

Deploy as a Vite project on Vercel. Use `npm run build` as the build command and `dist` as the output directory.
